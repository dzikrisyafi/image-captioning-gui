# image-captioning-gui
Image Captioning Application is an application to automatically generate captions on images using CNN and LSTM.
## Pre-Requisites
- Tensorflow 2.4
- Keras 2.4
- Numpy 1.19.3
- Ntlk 3.5
- Pickle 4.0
- Pillow 7.0.0
